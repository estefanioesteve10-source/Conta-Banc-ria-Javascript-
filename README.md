# Sistema Bancário em Node.js (CLI)

Este é um projeto simples de **Sistema Bancário em linha de comando (CLI)** desenvolvido em **Node.js**, utilizando o módulo `readline` para interação com o utilizador.

O sistema permite simular operações bancárias básicas, como:

- Criação de contas bancárias
- Geração automática de IBAN
- Depósitos
- Levantamentos
- Consulta de saldo e dados da conta
- Listagem de todas as contas criadas

Os dados são armazenados temporariamente em memória, sendo ideal para fins **didáticos**, **treino de lógica de programação** e **aprendizagem de Node.js**.

### Tecnologias Utilizadas
- Node.js
- JavaScript
- Módulo readline

### Objetivo
Este projeto tem como objetivo reforçar conceitos de:
- Estruturas de dados
- Funções
- Condições
- Interação com o utilizador via terminal
